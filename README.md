# Smart Online Wardrobe Planner

During my engineering studies, I decided to build an intelligent digital closet, a project designed to strengthen my backend development skills using Go. The system tracks user clothing inventories and provides recommendations based on real-time weather forecasts, seasons, occasions, and fashion trends.

## Tech Stack
* **Backend:** Go (Golang)
* **Database:** PostgreSQL (Relational)
